# Ninja Wars

Small project to see the differences between an OOP and functional approach to the same problem <https://github.com/Steelstone3/Thunders-Edge>.

## Running Ninja Wars

> cd ~/NinjaWars
>
> cargo build
>
> cargo run

## Tests

> cd ~/NinjaWars
>
> cargo test

## Dependencies

Follow the steps for installing rustc runtime for your given operating system.

> <https://www.rust-lang.org/tools/install>
